# Interactive Resume

A modern, interactive resume built with Next.js and Tailwind CSS.

## Features

- Light/Dark mode toggle
- Responsive design
- Interactive sections
- Portfolio showcase
- Contact form
- Certifications with viewable PDFs
- Profile picture

## Deployment

This project is deployed on Vercel.
